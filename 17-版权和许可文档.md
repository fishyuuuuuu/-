## 项目结构说明

### 后端项目 (seckill-go)
- **主要语言**: Go 1.25.1
- **框架**: Gin Web框架
- **数据库**: MySQL 8.0 + GORM ORM
- **缓存**: Redis 7
- **消息队列**: RabbitMQ 3-management
- **日志**: Zap日志库

### 前端项目 (seckill-vue)
- **主要语言**: JavaScript/TypeScript
- **框架**: Vue 3.5.18
- **构建工具**: Vite 7.0.6
- **路由**: Vue Router 4.5.1
- **HTTP客户端**: Axios 1.11.0
